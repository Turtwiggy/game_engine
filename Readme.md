An opengl game engine project. 

### Features

#### general
- threaded texture loading
- controller support (thanks SDL2)
- audio (thanks OpenAL)

#### 2d 
- sweep and prune collision detection for aabb
- batch renderer
- shadow casting

#### 3d
- model loading
